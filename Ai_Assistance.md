# AI assistance
In our Dinopet Walker project, we use AI assistance tools during development to facilitate certain steps and improve code quality.

## Interests of using 
1. Help in understanding the functioning and behavior of the Android system with an application.
2. Optimization and suggestions for code improvement.
3. Help in understanding and maintainability of the code.
4. Identify and correct certain bugs and unexpected behaviors. 
5. Apply best practices during development.

## AI assistance tools used 
We mainly used the following AI tools only via via a web interface :

1. Claude (Sonnet 4.6).
2. ChatGPT (GPT-5.3).

## How we interacted with AI
Our interaction with AI was through manual discussion and copy/paste. We did not use auto completion tools or agent based modes.

### process
- Asking questions or describing a problem.
- Paste part of our code if needed.
- Receiving suggestions or explanations.
- Copying back only the relevant parts into our project.


## Types of usage
We used AI for several purposes:

- Generating code (functions, appropriate parameters for a feature).
- Generating visual ideas (images, files in HTML format)
- Debugging and fixing errors
- Improving code structure and refactoring
- Writing and reformulating documentation
- Understanding concepts (Flutter, Android behavior, architecture)


## Integration of AI generated content
Our typical process was:

- Read and understand the generated code
- Keep only what is useful (some suggestions are copied, others serve as inspiration).
- Adapt the selected content to fit our project before committing


## Evolution of usage 
Our use of AI was not separated by phases, but rather depended on the features we were working on.

- At the beginning, we used it mainly to understand Flutter, Android behavior, and how to structure the project.
- We used it more frequently to help with features like animations and statistics.
- For specific tasks, we used it occasionally for things like Firebase setup, or calculations.
- For the mockups in Figma, we used it to get ideas from generated images or pages (in HTML format), and then we manually recreated the elements we liked (No AI tool was directly integrated into Figma to automatically generate the designs).
