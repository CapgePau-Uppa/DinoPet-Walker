# AI assistance
In our Dinopet Walker project, we use AI assistance tools during development to facilitate certain steps and improve code quality.

## Interests of using 
1. Help in understanding the functioning and behavior of the Android system with an application.
2. Optimization and suggestions for code improvement.
3. Help in understanding and maintainability of the code.
4. Identify and correct certain bugs and unexpected behaviors. 
5. Apply best practices during development.

## AI assistance tools used 
1. Claude
2. ChatGPT
